# SaaSKit Monorepo (vNext)

Fresh scaffold for the new modular SaaSKit platform.

## Structure

- `apps/app` – Vite + React product app (placeholder)
- `apps/web` – Astro marketing site (placeholder)
- `apps/api` – Fastify API server (placeholder)
- `packages/*` – Installable feature packages (`config`, `db`, `auth`, `validator`, etc.)
- `docs/` – Planning notes and architecture decisions

## Getting Started

1. Install pnpm (>=10)
2. Run `pnpm install` (after package manifests are added)
3. Use `pnpm create saaskit-app` (to be implemented) to scaffold optional packages
